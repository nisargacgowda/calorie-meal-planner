 **🥗 Calorie Meal Planner (Full Stack)**
A full-stack web application that generates personalized calorie-based meal plans with **Vegetarian, Non-Vegetarian, and Vegan** options — powered by a backend and MongoDB database.

---

## 🌟 About The Project  

This project helps users generate meal plans based on their calorie requirements and dietary preferences.  
Unlike basic static websites, this application uses a backend server and **MongoDB database** to manage dynamic data and improve scalability.
💡 Designed to demonstrate full-stack development skills.

---

## ✨ Features  

✅ Calorie-based meal plan generation  
🥬 Vegetarian option  
🍗 Non-Vegetarian option  
🌱 Vegan option  
🔄 Regenerate meal plan  
📱 Responsive design (Mobile + Desktop)  
📩 Contact Us page with feedback storage  
🗄️ MongoDB database integration

## ▶️ How To Run Locally  

1️⃣ Clone the repository  
2️⃣ Install dependencies  

```
npm install
```

3️⃣ Start the server  

```
node server.js
```

4️⃣ Open in browser:  

```
http://localhost:3000
```
